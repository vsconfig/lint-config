## Installation

1. Install the dependencies.

**npm:**

```bash
npm i -D eslint @vsconfig/eslint-config-typescript
```

**yarn:**

```bash
yarn add -D eslint @vsconfig/eslint-config-typescript
```

2. Create a `.eslintrc.*` file.

**JSON:**

```json
{
  "extends": ["@vsconfig/eslint-config-typescript"]
}
```

**JS:**

```js
module.exports = {
  extends: ["@vsconfig/eslint-config-typescript"]
}
```
