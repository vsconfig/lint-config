## Installation

1. Install the dependencies.

**npm:**

```bash
npm i -D prettier @vsconfig/prettier-config
```

**yarn:**

```bash
yarn add -D prettier @vsconfig/prettier-config
```

2. Configure your `package.json`.

```json
{
  "name": "my-cool-library",
  "version": "9000.0.1",
  "prettier": "@vsconfig/prettier-config"
}
```
