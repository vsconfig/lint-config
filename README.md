## Installation

### eslint-config

Eslint Config is available as an [npm package](https://www.npmjs.com/package/@vsconfig/eslint-config).

**npm:**

```bash
npm i -D eslint @vsconfig/eslint-config
```

**yarn:**

```bash
yarn add -D eslint @vsconfig/eslint-config
```
